# 🔔 Система уведомлений для одобрения заявок исполнителя

## 📋 Обзор

Реализована полная система уведомлений, которая отправляет пользователю уведомление сверху экрана при:
- ✅ **Одобрении заявки** на роль исполнителя
- ❌ **Отклонении заявки** на роль исполнителя
- (Аналогично для одобрения/отклонения песен)

## 🚀 Шаги внедрения

**Результат:**

- ✅ Обновлены RPC функции для отправки уведомлений

### 2. Компоненты Dart (уже готовы!)

#### Сервис уведомлений
```dart
// lib/data/services/notification_service.dart
```
- Получает уведомления с сервера
- Следит за новыми уведомлениями через Supabase Realtime
- Методы для пометки как прочитанные

#### Provider для состояния
```dart
// lib/presentation/providers/notification_provider.dart
```
- Управляет состоянием уведомлений
- Инициализируется при запуске приложения

#### UI для отображения
```dart
// lib/presentation/widgets/notification_overlay.dart
```
- **NotificationOverlay** — обёртка приложения для отображения уведомлений
- **_NotificationCard** — красивая карточка уведомления сверху экрана
- **NotificationBadge** — значок с количеством непрочитанных (опционально для app bar)

### 3. Как это работает

#### Процесс одобрения заявки:

```
[Admin в admin_panel нажимает "Одобрить"]
         ↓
[Вызывается ArtistApplicationService.approve()]
         ↓
[На сервере RPC функция approve_artist_application()]
         ↓
[Функция обновляет запись в artist_applications]
         ↓
[Функция присваивает роль "artist" пользователю]
         ↓
[⭐ Функция INSERT новое уведомление в таблицу notifications]
         ↓
[Приложение пользователя получает уведомление через Realtime]
         ↓
[На экране пользователя появляется уведомление сверху 🎉]
```

#### Структура уведомления в БД:

```sql
{
  id: "uuid",
  user_id: "uuid пользователя которому одобрили",
  type: "artist_application_approved",
  title: "Заявка одобрена",
  message: "Поздравляем! Ваша заявка на роль исполнителя одобрена...",
  data: {
    "application_id": "uuid",
    "artist_name": "названи проекта",
    "reviewer_note": "комментарий модератора"
  },
  read: false,
  created_at: "timestamp"
}
```

## 📱 Как это выглядит

### На экране пользователя:

1. **Когда приходит уведомление:**
   - Появляется зеленая карточка сверху экрана (для одобрения) 
   - Содержит название и описание уведомления
   - Автоматически исчезает через 5 секунд
   - Можно закрыть кликом на крест

2. **Иконки в app bar (опционально):**
   - Добавить `NotificationBadge` в AppBar
   - Будет показывать красный значок с количеством непрочитанных

## 🔧 Дополнительные функции

### Получить все уведомления:
```dart
final provider = context.read<NotificationProvider>();
final allNotifications = provider.notifications;
final unreadCount = provider.unreadCount;
```

### Пометить как прочитанное:
```dart
await provider.markAsRead(notificationId);
```

### Пометить все как прочитанные:
```dart
await provider.markAllAsRead();
```
